API http+json desenvolvida em django rest framework usando sqlite.

GET:
    http://servidor/item/RFIDCODE/
    ex: http://servidor/item/315748
     { Nome: "Leite 1L", Preco: "299", Peso: 1020 }
    http://servidor/compra/
     { [ {Nome: "Leite 1L", Preco: "299", Quantidade: 3 }, ... ] }
    http://servidor/carrinho/QRCODE  - Pode retornar 404 (indica que o carrinho n√£o esta selecionado)
     { IDCompra: 719, Jaqueta: 3 }

POST:
    Adicionar comprador
    http://servidor/cliente/
     { Nome: "Jose da Silva", email: "josesilva@gmail.com", senha: "12345" }

    Iniciar compra
    http://servidor/compra
     { email: "josesilva@gmail.com", senha: "12345", QRCode: 2618271, Jaqueta: 3 }

PUT:
    Modificar/Adicionar/Remover Item da compra
    http://servidor/compra/IDCOMPRA/RFIDCODE
    ex: http://servidor/compra/719/315748
     { Quantidade: 3 }
     { Quantidade: 0 }

DELETE:
    Abortar compra
    http://servidor/compra
     { email: "josesilva@gmail.com", senha: "12345" }

    Terminar compra pagando
    http://servidor/compra
     { email: "josesilva@gmail.com", senha: "12345", pagamento: 918203817 }

    Avisar que o carrinho saiu e a compra pode terminar
    http://servidor/carrinho/QRCODE
     { }
